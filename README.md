# Kirill Content Technologies

Public, sanitized repository for Kirill's content production technologies.

This is a system for turning daily signals into multi-platform content:

- AtlasRepo and open-source project radar;
- want2view video trend intelligence;
- YouTube Shorts and long-form scripts;
- TikTok/Reels scene plans;
- Instagram carousels;
- X / Threads posts;
- article and SEO content systems;
- creator/avatar video experiments;
- social distribution agents;
- Russian article factories for other niches;
- TrendVi / want2view content-factory and SEO technology slices;
- broader local producer / article / social agent source samples;
- niche research exports;
- partner screencast workflows.

- `17_niche_research_and_exports/` - selected niche research and channel/export materials.

## Important

This repository is sanitized for public sharing.

Do not commit:

- `.env`
- API keys
- Telegram sessions
- browser profiles
- real Postiz credentials
- local state
- logs with tokens

Use fresh local credentials outside the repo.

Any historical credentials that may have existed in local working docs should be rotated before using this system in production.
