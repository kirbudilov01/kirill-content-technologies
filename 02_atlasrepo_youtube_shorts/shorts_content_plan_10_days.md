## День 1: Open-source вместо платных SaaS

### 1. Хватит платить за 5 AI-инструментов

Референс: Kevin Stratvert, `Stop Paying for 5 AI Tools (Use This Instead)`, 3.5M views.

Хук: "Хватит платить за пять AI-сервисов. Часть из них уже заменяется бесплатными GitHub-репозиториями."

Сценарий:
1. "У тебя есть отдельный сервис для заметок, отдельный для автоматизации, отдельный для поиска по файлам."

Скринкаст для партнера: показать GitHub search с запросами `open source alternative notion`, `open source automation ai`, `self hosted ai search`; затем 3 вкладки с README популярных repo, звезды, install block, demo gif.
