# ATLAS REPO — Shorts Voiceover V3, vibe-code edition

Цель: сценарии должны быть понятны с первой секунды даже зрителю, который не знает Hermes/OpenClaw/MCP. Тон: живой builder/vibe-coding, меньше терминологии, больше “что это дает мне прямо сейчас”.

## Правила v3

- Первая фраза всегда дает контекст простым языком.
- Не говорим “репозиторий позволяет”. Говорим “ты быстрее собираешь”, “агент получает руки”, “workflow меньше надо собирать руками”.
- В каждом ролике есть простой вердикт: save / test / skip.
- Atlas Repo — не реклама в лоб, а радар: “я собираю такие штуки, чтобы быстрее видеть что реально полезно”.
