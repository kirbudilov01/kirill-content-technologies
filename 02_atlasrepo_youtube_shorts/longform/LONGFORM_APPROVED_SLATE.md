# Long-form approved slate

Железно утвержденная пачка длинных видео. Пока работаем вокруг этих тем: подбираем реальные repo, paid competitors, демо, скринкасты, тезисы и потом пишем сценарии.

## Approved topics

### 1. Я собрал micro-SaaS за вечер только из open-source repo

**Статус:** approved.

**Почему берем:** продукт, скорость, open-source, деньги. Понятно широкой аудитории.

**Что нужно добить:** выбрать нишу SaaS и repo-кирпичи.

**Партнеру:** предложить 2-3 ниши и подобрать под каждую frontend starter, auth, database, UI, AI/research, automation, deploy. Снять GitHub README, demo, setup, итоговый экран продукта.

---

### 2. Я проверил 100 AI-инструментов: 10 реально стоят твоего времени

**Статус:** approved.

**Почему берем:** сильный конкурентный фрейм, продает экономию времени.

**Что нужно добить:** список 100 инструментов и финальные 10.

**Партнеру:** собрать инструменты из AtlasRepo/нашей базы/GitHub, отобрать 10, снять по каждому сайт/GitHub, README/demo, один proof, одно ограничение, verdict `save/test/skip`.

---

### 3. Я открыл 50 новых GitHub repo за день. Большинство мусор, но эти 7 — золото

**Статус:** approved.

**Почему берем:** хорошая драматургия: шум, отбор, золото.

**Что нужно добить:** категории repo и критерии отбора.

**Партнеру:** собрать 50 свежих repo по AI agents, MCP, automation, devtools, content/marketing, SaaS alternatives. Снять быстрый монтаж слабых repo и proof-экраны 7 победителей.

---

### 4. Хватит платить за 5 AI-сервисов: вот open-source замены

**Статус:** approved.

**Почему берем:** деньги и подписки понятны всем.

**Что нужно добить:** конкретные пары paid tool -> open-source alternative.

**Партнеру:** подобрать 5 платных сервисов и open-source альтернативы. Снять pricing, GitHub/сайт альтернативы, README/demo, где заменяет, где уступает.

### 5. Перед тем как купить очередную AI-подписку, проверь эти 10 GitHub repo

**Партнеру:** собрать repo по research, automation, content, UI/design, devtools, docs, agents, browser automation, analytics, knowledge base. Снять quick proof по каждому.

### 11. AI ломает SaaS-бизнес: что будет с обычными подписками

**Статус:** approved, но только через конкретные примеры.

**Почему берем:** большая рыночная тема, может стать сильным философско-практическим роликом.

**Что нужно добить:** 3-5 категорий, где AI/open-source реально давит SaaS.

**Партнеру:** собрать примеры paid SaaS vs open-source/AI alternatives: support, analytics, automation, content, devtools. Снять pricing, demo, ограничения, итоговую карту рынка.

## Not in approved slate for now

- `AI сам собрал мне контент-план на месяц...` — хорошая тема, но сейчас не вошла в утвержденную пачку.
- `Я протестировал AI tools для контента...` — тоже норм, но пока не в железном списке.
- `AI Tools Tier List 2026...` и `Open-source AI tools: S-tier...` — возможны позже, но сейчас не среди выбранных номеров.
- Все Cursor / beginner explainers / generic news темы — не берем.
