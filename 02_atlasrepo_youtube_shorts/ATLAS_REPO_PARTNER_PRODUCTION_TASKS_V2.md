# ATLAS REPO — Partner Production Tasks V2

Документ для партнера. Задача партнера: выгрузить proof materials и снять screencast так, чтобы у ведущего и монтажера был материал для удержания: proof, demo, limitation, verdict.

## Global shooting rules

- Desktop long-form: 16:9, крупный zoom, читаемый текст.
- Shorts cutaways: 9:16 или 16:9 с учетом crop.
- Не тратить больше 10 минут на запуск одного demo. Если не пошло — docs/screens/issues/mock.
- Всегда снимать limitation proof, иначе ролик будет рекламным.
